发表于 2017-02-08   |   分类于 工具   |

gulp.task()

  • gulp.task() 方法用来定义任务，其语法为：

  • name 为任务名

  • deps 一个数组用来定义当前任务需要依赖的其他任务。当前任务会在所有依赖的任务执行完毕后才开始执行。如果没有依赖，则可省略这个参数。

  • fn 为任务函数，任务代码都写在里面。该参数也是可选的。

  • 多个任务执行

    • gulp中输入一行指令运行一系列任务，可以通过配置任务依赖来实现。
    • 含有异步操作该如何顺序执行任务呢？
      • 第一：在异步操作完成后执行一个回调函数来通知gulp这个异步任务已经完成，这个回调函数就是任务函数的第一个参数。
      • 第二：定义任务时返回一个流对象。适用于任务就是操作gulp.src获取到的流的情况。
      • 第三：返回一个Promise对象。

gulp.watch()

  • gulp.watch()用来监视文件的变化，当文件发生变化后，我们可以利用它来执行相应的任务。

  • glob 为要监视的文件匹配模式，规则和用法与gulp.src()方法中的glob相同。

  • opts 为一个可选的配置对象，通常不需要用到
  • tasks 为文件变化后要执行的任务，为一个数组 cb 为一个函数。每当监视的文件发生变化时，就会调用这个函数。
